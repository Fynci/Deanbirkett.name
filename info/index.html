Hi! I'm Dean, a User Experience Designer currently working for Europeana in The Netherlands.

About me

I’m a passionate advocate of mobile first responsive web design, and a lover of accessible universal design. I have a broad range of UX skills, and apply them from inception, all the way through the various stages of a User Centred Design process.

An example: I’m brought in for stakeholder meetings, interview users, run sketching workshops, perform usability tests & card sorts, wireframe, prototype, create visual elements & mockups, and work with developer colleagues to implement the designs.

I also code things myself - this site is one of them.

About this site

To optimise the site’s load time, I opted to build it in Jekyll with all posts written in Markdown. The theme has been adapted from Balzac.

The logo and brand is by Micipedia, the font used throughout is Brandon Grotesque and the icon set is available at Entypo.

I added Fitvids.js to handle responsive video and reading_time.rb is added to estimate reading times of articles.
